# Nervos.rb

Nervos AppChain Ruby SDK

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'nervos.rb', github: "cryptape/nervos.rb"
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nervos.rb
